🔧 EXTENDING THE SYSTEM
================================================================================

Want to add a new Python topic?

1. Open: python_tutor.py

2. Find: PYTHON_TOPICS dictionary

3. Add:
   "generators": {
       "level": SkillLevel.INTERMEDIATE,
       "keywords": ["generator", "yield"],
       "explanation": {
           "beginner": "...",
           "intermediate": "...",
           "advanced": "..."
       },
       "code": "...",
       "why_it_works": "...",
       "common_mistakes": [...],
       "best_practices": [...]
   }

4. Add to QUESTION_PATTERNS:
   "generator": "generators"

5. Restart Flask app

Done! Now "What is a generator?" works ✅

================================================================================
🆘 TROUBLESHOOTING
================================================================================

Q: App won't start
A: Run: pip install flask
   Then: python app.py

Q: Port 5000 in use
A: Run on different port by editing app.py:
   app.run(debug=True, port=5001)

Q: Python tutor not responding
A: Question might not match keywords
   Try: "What is a loop?"

Q: Code not highlighting
A: Browser should load Highlight.js automatically
   Check browser console for errors

Q: Questions not being recognized
A: Add more keywords to QUESTION_PATTERNS in python_tutor.py
